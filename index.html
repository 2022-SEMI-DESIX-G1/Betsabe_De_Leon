¿Qué es Javascript y en qué se asemeja a Java?

JavaScript (JS) es un lenguaje de programación ligero, interpretado, o compilado justo-a-tiempo (just-in-time) con funciones de primera clase. Ambos lenguajes usan como base a C, pero no son similares ni están relacionados. Java se utiliza en la creación de aplicaciones Android o de escritorio, y JavaScript se utiliza principalmente para hacer que las páginas web sean más interactivas.
